# Crypto Investigation Case Study

## Project Summary

This project is a professional blockchain investigation case study using Etherscan to analyze wallet activity, transaction behavior, contract interactions, token transfers, and suspicious on-chain patterns.

The goal of this project is to demonstrate crypto intelligence, blockchain analysis, wallet tracing, and professional investigation-style documentation.

---

## Problem Statement

Crypto investigations require analysts to understand wallet behavior, transaction flows, smart contract interactions, token transfers, and on-chain evidence.

Many blockchain users and investigators struggle to turn raw wallet activity into a clear, professional report. This project shows how blockchain activity can be reviewed, organized, and documented in a structured case-study format.

---

## Investigation Workflow

### 1. Select Target

The investigation begins with one target, such as:

- Wallet address
- Transaction hash
- Token contract
- Suspicious dApp contract
- Address connected to a scam, exploit, or suspicious launch

### 2. Review Wallet Profile

The wallet is reviewed for:

- ETH balance
- Token holdings
- First and last transaction activity
- Number of transactions
- Labels or known address tags
- Contract interactions
- Wallet behavior patterns

### 3. Analyze Transactions

Transactions are reviewed for:

- Incoming transfers
- Outgoing transfers
- Failed transactions
- Internal transactions
- Token transfers
- Contract calls
- Function names
- Transaction values
- Gas usage
- Timestamp patterns

### 4. Review Contract Interactions

Contract interactions are reviewed to understand:

- What protocol or contract was used
- Whether the wallet approved token spending
- Whether swaps, transfers, claims, mints, or contract calls occurred
- Whether interaction behavior looked normal or suspicious

### 5. Document Findings

Findings are organized into a case-study format with:

- Executive summary
- Target overview
- Transaction timeline
- Wallet behavior notes
- Risk indicators
- Supporting evidence
- Final conclusion
